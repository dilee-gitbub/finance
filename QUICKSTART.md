## 🛠️ 트러블슈팅

### 문제: "API 키가 없어요"
**해결**: `.env.local` 파일에 올바른 API 키를 입력했는지 확인

### 문제: "포트 5173 이미 사용 중"
**해결**:
```bash
npm run dev -- --port 5174
```

### 문제: "회사 데이터가 로드 안 됨"
**해결**:
```bash
npm run parse-xml
```

### 문제: "차트가 안 보여요"
**해결**: 브라우저 개발자 도구 (F12) → Console에서 에러 확인
