🌐 배포 (Vercel)
================================================================================

1. GitHub 저장소 생성
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <GITHUB_URL>
   git push -u origin main

2. Vercel 연결
   - vercel.com 접속
   - GitHub 저장소 선택
   - 자동 배포 활성화

3. 환경 변수 설정 (Vercel 대시보드)
   OPENDART_API_KEY=...
   GEMINI_API_KEY=...

4. Deploy!

자세한 가이드: DEPLOYMENT.md 참고
